# Hyperreal

Welcome to **Hyperreal** — a documentation site built with
[Nextra](https://nextra.site) and the docs theme.

This project uses the default **content directory** convention with static
generation at build time.

## What you get

- Docs theme with sidebar navigation
- MDX content in the `content/` directory
- Static export for GitHub Pages
